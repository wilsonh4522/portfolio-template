CYOA Project

In class we did a CYOA (Choose Your Own Adventure) Project. It is where we took our skills we learned in class. Which is learning how to do command lines and git. We took these skills and we made a project where we chose a theme and make situations and it would lead of to another. We also had to work with a partner

For this project we did a theme about "The Hunger Games" from the movie. Me and my partner chose this theme because it had many situations and possibilities we could chose from. During this project I had some challenges . For example it was hard to choose the situations since there were so many possibilities I could chose from. Another challenge was communication since my teammate switched to a different class and it was hard to tell him what I wanted. One takeaway I have for this project is communication is key for teamwork and that making a plan before you do a project is helpful since it will help you stay organized.
