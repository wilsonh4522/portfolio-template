SEP11 Freedom Project

In SEP11 we learned about JavaScript and did a year long project where we would learn a tool and create anything we wanted

I first had to come up of what I wanted to create. So I chose to make a Hangman game. The user had to guess the letters of the word to get the whole word. The tool I chose is KaboomJS. It is a JavaScript game programing libary that I used to create my game. Every Monday during class we would work on our project until we got the MVP (Minimal Viable Product). For my beyond MVP I planned to add features in the game. For example the user could chose 2 different levels of hangman, diplay the inccorect words, and make the actual hangman stick figure each time the user gets it wrong

After we finished our project we had to present to the class. I made a slideshow about my project that would explain my thoughts, challenges, and takeaways when making the projects.

After presenting to the class we had to present to other people. We had an expo where people would walk around and see our project and there were also judges to rate our project. Because of this we had to create and elevator pitch that would explain our project and what it was about.

Making this project was challenging and involved a lot of progress, hardwork, and thinking, becuase of this I made a blog that explains my thinking, challenges, progress, and solutions.
